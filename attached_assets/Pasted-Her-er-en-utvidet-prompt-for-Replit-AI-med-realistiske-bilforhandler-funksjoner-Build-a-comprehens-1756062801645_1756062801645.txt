Real Dealership Contract Types:

Kjøpekontrakt (Purchase Agreement)

Customer buys car from dealership
Includes: Car details, price, delivery date, warranty terms
Status flow: Tilbud → Signert → Betalt → Levert


Innbyttekontrakt (Trade-in Agreement)

Customer trades old car + cash for new car
Includes: Trade-in valuation, condition report, settlement calculation
Status flow: Vurdering → Avtalt → Innlevert


Finansieringsavtale (Financing Agreement)

Bank/finance company contracts
Integration with financing partners
Status flow: Søknad → Godkjent → Signert → Utbetalt


Leasingkontrakt (Lease Agreement)

Long-term rental agreements
Monthly payment tracking
Status flow: Tilbud → Godkjent → Aktiv → Avsluttet


Servicekontrakt (Service Agreement)

Extended warranty, maintenance packages
Recurring service intervals
Status flow: Solgt → Aktiv → Brukt → Utløpt

Sample Enhanced Data Structure:
typescriptinterface DealershipContract {
  id: string;
  type: 'purchase' | 'trade-in' | 'financing' | 'lease' | 'service';
  contractNumber: string;
  
  // Customer
  customer: {
    name: string;
    phone: string;
    email: string;
    personnummer: string;
  };
  
  // Car being sold
  vehicle: {
    brand: string;
    model: string;
    year: number;
    vin: string;
    price: number;
    status: 'in-stock' | 'ordered' | 'arrived' | 'delivered';
    deliveryDate?: Date;
  };
  
  // Trade-in (if applicable)
  tradeIn?: {
    brand: string;
    model: string;
    year: number;
    regNr: string;
    valuedAt: number;
    condition: string;
    received: boolean;
  };
  
  // Financial
  financial: {
    totalPrice: number;
    downPayment: number;
    tradeInValue: number;
    financedAmount: number;
    paymentMethod: 'cash' | 'financing' | 'lease';
    bankName?: string;
    approved?: boolean;
  };
  
  // Workflow
  status: 'offer' | 'signed' | 'financed' | 'paid' | 'ready' | 'delivered';
  salesPerson: string;
  commission: number;
  requiredDocuments: string[];
  completedDocuments: string[];
  nextAction: string;
  createdAt: Date;
  deliveryDate?: Date;
}
